## 📌 這個程式在做什麼？

本系統整合 **5 大量化指標**，每天自動計算你指定的台股，用「紅綠燈」的方式告訴你：

| 燈號 | 意思 | 建議動作 |
|------|------|---------|
| 🟢 綠燈 | 多方訊號強，多項指標同時看漲 | 考慮買進 |
| 🔴 紅燈 | 空方訊號強，多項指標同時看跌 | 考慮賣出 |
| 🟡 黃燈 | 多空訊號不明確，市場橫盤觀望 | 持股等待 |

同時還提供：
- 📊 **歷史 K 線圖**，直接標出所有買進 🟢 與賣出 🔴 的時間點
- 📈 **策略 vs 放著不動** 的模擬投報率比較
- 🛡️ **上班族低頻模式**，自動過濾盤整區雜訊，大幅減少頻繁交易

---

## 🔬 五大量化指標說明

| # | 指標名稱 | 看什麼 | 大白話解讀 |
|---|---------|--------|-----------|
| 1 | 趨勢方向 (Trend) | 20 日均線方向 | 股價有沒有站在均線上，是漲勢還是跌勢 |
| 2 | 布林通道 (Space) | 布林帶位置 | 現在買是便宜還是太貴，是在下軌還是上軌 |
| 3 | MACD 動能 (Momentum) | 能量柱增減 | 上漲力道強不強，有沒有在加速 |
| 4 | 成交爆量 (Force) | 量能比較 | 有沒有主力大買或大賣，買氣熱不熱 |
| 5 | 法人籌碼 (Chip) | 外資 + 投信 | 大法人有沒有連續買進撐盤 |

每項指標：**多方 +1 分 / 中性 0 分 / 空方 -1 分**  
加權合計分數 **≥ +3 分 → 綠燈買進；≤ -3 分 → 紅燈賣出**

## 🖥️ 如何使用這個程式

### Step 1：輸入想查的股票

在頁面上方的搜尋欄輸入：
- 股票代碼（如 `2330`、`8069`）
- 或中文名稱（如 `台積電`、`鴻海`、`元太`）

程式會自動比對並載入最近一年的數據。

---

### Step 2：看今天的燈號

頁面最上方的大卡片會顯示：
- 當前多空燈號（🟢 / 🔴 / 🟡）
- 今日綜合評分
- 最新資料日期與收盤價

---

### Step 3：看 K 線圖上的買賣點

K 線圖上：
- 🟢 **▲ 買 xxx 元** = 策略模擬的買進點
- 🔴 **▼ 賣 xxx 元** = 策略模擬的賣出點
- 亮藍色成交量 = 當日法人合力大買護盤

> 💡 手機版可展開圖表下方的「**📋 歷史交易訊號明細**」查看每次交易的詳細日期與損益。

---

### Step 4：調整左側「策略因子比重設定」

你可以透過左側滑桿，手動調整 5 個指標的權重（0 到 5 倍），  
或點擊 **「🤖 一鍵自動最佳化比重」** 讓 AI 自動幫你找出最佳比重配置，  
讓策略報酬最大化超越單純持有。

---

### Step 5：開啟「上班族低頻交易模式」（推薦！）

在左側「🛡️ 濾網設定」區塊開啟低頻模式後，系統將自動啟用：

| 過濾器 | 說明 |
|--------|------|
| ① 訊號連續確認天數 | 訊號需連續出現 N 天才執行，預設 2 天，濾掉假突破 |
| ② 最低持有天數 | 買進後至少持有 N 天才能賣，預設 5 天，避免盤整洗盤 |
| ③ 布林擠壓偵測過濾 | 布林帶極度收窄時自動抑制買進，避免橫盤套牢 |

## 📦 本機安裝與執行

### 環境需求

- Python 3.10 以上
- pip

### 安裝步驟

```bash
# 1. 複製此倉庫
git clone https://github.com/YOUR_USERNAME/taiwan-stock-signal.git
cd taiwan-stock-signal

# 2. 安裝相依套件
pip install -r requirements.txt

# 3. 啟動程式
streamlit run app.py
```

啟動後在瀏覽器開啟 👉 **http://localhost:8501**
